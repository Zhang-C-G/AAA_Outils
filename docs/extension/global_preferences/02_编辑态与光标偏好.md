# 全局私人偏好：编辑态与光标偏好
最近同步：`2026-04-25`

## 1. 定位

本文件记录进入编辑态时的通用偏好。

## 2. 具体规则

### 2.1 原位编辑优先

- 如果某结构支持直接改名、直接编辑，优先采用原位编辑
- 不优先使用“点一下后整个控件突然换形态”的实现

### 2.2 尺寸稳定优先

- 进入编辑态、切换态、展开态时，优先保持区域尺寸稳定
- 不喜欢明显的外框突变、尺寸突变、控件替换感

### 2.3 编辑态不默认全选

- 进入重命名或编辑态时，不默认把整段内容全选
- 优先采用“直接聚焦，光标放在文本末尾”的方式

说明：

- 默认全选虽然方便覆盖输入，但也容易导致误删
- 对标题、栏目名这类短文本，当前更偏好在原内容基础上继续微调
