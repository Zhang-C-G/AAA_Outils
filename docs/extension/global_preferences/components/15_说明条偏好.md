# 全局私人偏好：说明条偏好

最近同步：`2026-04-26`

## 1. 文档定位

适用于快捷键说明条、同步说明条、只读提示条、顶部信息条这类“非编辑、非主操作”的辅助说明区域。

## 2. 默认偏好

### 2.1 视觉

- 说明条允许使用斜杠纹理背景
- 斜杠纹理应是低对比、深色、克制的，不做花哨装饰
- 说明条整体应保持扁平、横向展开、易扫读

### 2.2 作用

- 说明条主要用于承载规则、快捷键、同步状态、只读提示
- 它是辅助层，不应抢主内容的视觉优先级
- 允许比普通输入框多一点识别度，但不应做成按钮感

### 2.3 边界

- 说明条与输入区、编辑区应有清楚区分
- 不把说明条做成可编辑控件的样子
- 如果使用斜杠背景，应优先用于“说明性区域”，而不是表单主体

## 3. 禁止倾向

- 不喜欢纯色死板提示条，缺少层级感
- 不喜欢把斜杠纹理铺满大面积主体区域
- 不喜欢说明条过亮、过花，抢走主操作注意力
