# 全局私人偏好：标签页偏好

最近同步：`2026-04-26`

## 1. 文档定位

适用于模式标签、页签、同级视图切换入口。

## 2. 默认偏好

### 2.1 结构

- 标签页是同级视图切换，不应混入异质动作按钮
- 切换类入口尽量集中在同一层，不拆得到处都是

### 2.2 激活态

- 当前激活标签要一眼能认出
- 未激活态应清楚可点，但不应压过激活项

### 2.3 命名

- 标签命名应尽量直观
- 名称应反映模块真实职责，而不是技术实现细节

### 2.4 扩展

- 新增标签时应优先保持整体层级一致
- 不应为了某个局部需求把标签页变成半菜单半工具栏的混合结构

## 3. 禁止倾向

- 不喜欢同级入口长得像不同体系
- 不喜欢激活态和未激活态差异过弱
- 不喜欢在标签条里混入与切换无关的杂项动作
