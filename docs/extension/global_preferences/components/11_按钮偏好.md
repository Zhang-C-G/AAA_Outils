# 全局私人偏好：按钮偏好

最近同步：`2026-04-26`

## 1. 文档定位

适用于普通按钮、主按钮、危险按钮、图标按钮以及模式切换按钮。

## 2. 默认偏好

### 2.1 主次分层

- 页面里应有明确主按钮，但数量要克制
- 次按钮不应抢走主动作视觉优先级
- 不要把所有按钮都做成主按钮风格

### 2.2 稳定性

- 按钮尺寸应稳定
- 激活、悬停、禁用状态要清晰
- 状态变化应明确但克制，不做过度跳动

### 2.3 文案与入口

- 高频按钮应尽量直接放在用户主路径上
- 危险按钮文案要直接，不绕弯
- 图标按钮若语义不够直观，应有提示

### 2.4 危险操作

- 危险按钮应与普通按钮有明确区分
- 但危险色应聚焦在危险动作本身，不把整片区域染成高风险视觉噪音

## 3. 禁止倾向

- 不喜欢主次不分
- 不喜欢为了“显眼”把按钮做得过于浮夸
- 不喜欢高频按钮藏得太深
