# 全局私人偏好：自动保存与保存按钮偏好
最近同步：`2026-04-25`

## 1. 定位

本文件记录持续编辑型模块在保存链路上的全局偏好。

## 2. 具体规则

### 2.1 自动保存优先于手动保存按钮

- 对持续编辑型模块，优先自动保存
- 手动保存按钮不是默认必选项

### 2.2 保留按钮不是目的，保存闭环才是目的

- 如果模块已经有稳定自动保存链路，就不优先再保留显式保存按钮
- 是否保留保存按钮，取决于真实闭环是否稳定，而不是取决于习惯性堆按钮

### 2.3 自动保存必须带兜底

- 自动保存不是“表面自动”，而是要有刷新、切换、异常时的可靠兜底
- 如果没有这些兜底，就不应轻易删掉保存入口
