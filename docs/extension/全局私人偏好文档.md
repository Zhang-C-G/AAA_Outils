# 全局私人偏好文档
最近同步：`2026-04-25`

## 1. 文档定位

这份文档现在是“全局私人偏好体系”的总索引。
它不再承担承载全部细则的职责，而是负责：

- 说明这套全局私人偏好文档是干什么的
- 说明进入第二阶段时该怎么读
- 给出分类结构和跳转入口
- 约束后续新增私人偏好时应写到哪里

也就是说：

- 本文件负责“总纲和索引”
- 具体私人偏好细则拆到子文档里维护

## 2. 使用方式

每次进入第二阶段时，建议按这个顺序读取：

1. `docs/extension/AI_DEVELOPMENT_PLAYBOOK.md`
2. 对应模块主文档 `docs/modules/*.md`
3. 该模块本次的标准能力说明文档
4. 本文件 `docs/extension/全局私人偏好文档.md`
5. 再按需要读取对应主题子文档
6. 最终融合为该模块的最终能力说明文档

## 5. 维护规则

后续新增全局私人偏好时，不要直接无序堆到本文件里。

推荐规则：

1. 先判断它属于哪一类主题
2. 写入对应子文档
3. 如果现有主题不适合，再新增新的主题子文档
4. 最后只在本文件补索引，不在本文件重复展开长篇细节
